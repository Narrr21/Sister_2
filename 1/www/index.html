Welcome to Assembly HTTP Server!

This HTTP server is implemented entirely in x86-64 Assembly language using Linux system calls.

✨ Features

🌐 TCP Socket Listening

Server listens on port 8080 using Linux socket system calls

🔄 Process Forking

Each client request is handled by a separate child process

📋 HTTP Methods

Supports GET, POST, PUT, and DELETE HTTP methods

📁 Static File Serving

Serves HTML, CSS, JavaScript, and image files

🛣️ Simple Routing

Routes requests based on URL path

🎯 MIME Type Detection

Automatic content type detection based on file extension
