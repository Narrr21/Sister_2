About This Project

This project is a proof-of-concept web server built from scratch using NASM assembly on a 64-bit Linux system.

Core Features

  • HTTP GET & POST: Handles basic web requests.
  • Static File Serving: Serves HTML, CSS, and JS files.
  • Form Handling: Processes `POST` requests from HTML forms.
  • Template Rendering: A simple find-and-replace templating system to inject dynamic content into a master layout.
